# AE2 WCWT

AE2 WCWT（Wireless Comprehensive Work Terminal）是一个面向 **Applied Energistics 2 / NeoForge 1.21.1** 的综合型无线终端模组。  
它的目标不是只做“无线合成终端”，而是把多个 AE2 生态附属模组中常用、常切换、常要开很多界面的功能，集中整合到一把无线终端里。

当前版本：`v1.1.0`

## 特性概览

- 18 列主仓库显示区，更大的无线物品查看面积
- 内置样板缓存区，便于批量编码、复制、替换、上传与整理
- 同时支持合成样板、处理样板、锻造台样板、切石机样板
- 集成高级编码、样板管理、工具包、卡槽箱等扩展 UI
- 支持快捷键直接打开不同扩展界面
- 支持多种可选模组联动，按已安装模组动态显示对应按钮或界面

## 主界面说明

WCWT 主界面可以理解为“增强版无线工作终端”，主要分为以下区域。

### 1. 主仓库显示区

- 用于查看 ME 网络中的物品、流体和其它类型内容
- 顶部有 4 个小按钮：
  - 物品显示
  - 流体显示
  - 其它类型显示
  - 若安装 `Extreme Sound Muffler`，会显示静音按钮

### 2. 顶部功能按钮

右上角一组小按钮，具体是否显示取决于终端配置和已安装模组：

- 无线终端设置
  - 调整 `Pick Block / 缺少时合成 / Restock / Magnet / Pickup To Me` 等选项
- 磁卡菜单
  - 依赖 `AE2WTLib` 的磁卡功能
- 垃圾桶
  - 打开终端内置垃圾菜单
- 静音
  - 仅在安装 `Extreme Sound Muffler` 时显示

### 3. 手动合成区

- 提供传统 3x3 手动合成网格
- 有“合成网格锁定”按钮

作用：

- 未锁定时，JEI/EMI 的编码区填充优先进入样板编码区
- 锁定后，JEI/EMI 的工作台配方优先进入手动合成 3x3

### 4. 样板编码区

支持 4 种样板模式：

- 合成样板
- 处理样板
- 锻造台样板
- 切石机样板

常用按钮：

- 编写样板
- 清空
- 合并材料开关
  - 处理样板模式下可控制“相同输入是否自动合并”
- 替代/流体替代开关
- 处理样板输出切换

### 5. 样板倍增区

位于编码区旁边，用于快速倍增/缩放当前样板内容。

默认提供：

- `x2`
- `x3`
- `x5`
- `=1`
- `/2`
- `/3`
- `/5`
- 以及额外一枚控制按钮

用途：

- 快速放大配方批量
- 快速还原成单份配方

### 6. 样板缓存区

- 用于暂存已编码或待处理的样板
- 很多扩展功能都以这里作为输入来源

支持的常见操作：

- 选中样板进行高级编码
- 批量上传到供应器
- 快速复制/替换
- 作为谐振/过载转换的输入来源

## 样板管理区

这一块主要对齐 `ExtendedAE / ExtendedAE Plus` 的样板管理体验。

### 核心功能

- 搜索供应器
- 搜索映射管理
- 样板供应器列表显示
- 显示模式切换
- 搜索模式切换
- 是否显示槽位切换
- 启用/关闭“自动上传样板”
- 直接打开供应器对应 UI
- 高亮定位供应器

### 样板管理按钮说明

- 增加映射
  - 将搜索关键字映射到指定供应器名
- 重载映射
  - 重新加载映射表
- 删除映射
  - 删除指定映射
- 取消
  - 清空输入框
- 显示模式
  - 全部 / 可见 / 未满
- 显示槽位
  - 切换是否显示供应器内部槽位
- 启用上传样板
  - 控制编码完成后是否优先尝试自动上传
  - 该开关现在会保存在 **终端物品自身**
- 搜索模式
  - 按输入 / 输出 / 输入输出搜索

### 供应器行按钮

每个样板供应器右侧会有若干按钮：

- 上传按钮
  - 将样板缓存区中的样板按顺序保存到该供应器
- 打开 UI 按钮
  - 尝试打开该供应器对应的目标机器界面
- 高亮按钮
  - 高亮定位供应器所在位置

### Shift 快速操作

可通过服务端配置控制是否启用：

- `Shift + 点击样板管理区样板`
  - 快速取出到物品栏
- 打开终端时，配合快捷逻辑可快速插入到第一个供应器

## 扩展 UI 按钮与界面

主界面右侧有一列扩展 UI 按钮。  
其中有些按钮始终存在，有些只有安装对应模组后才显示。

### 1. 高级编码

始终显示。

主要整合：

- `AdvancedAE` 的高级样板编辑思路
- WCWT 自己的样板缓存选中逻辑
- AE2 元件工作台相关功能

包含内容：

- 样板输入方向编辑区
  - 选中缓存区样板后可编辑输入方向
- 复制样板
  - 消耗空白样板复制所选样板
- 批量替换
  - 在缓存区内批量替换输入/输出内容
- 元件编辑区
  - 分区存储
  - 清空配置
  - 复制模式切换
  - 元件升级槽浏览

### 2. 装饰盔甲

仅在安装 `Cosmetic Armor Reworked` 时显示。

功能：

- 打开装饰盔甲面板
- 显示并操作装饰头盔、胸甲、护腿、靴子槽位

### 3. 饰品栏

仅在安装 `Curios` 时显示。

功能：

- 浏览 Curios 实际槽位
- 支持滚动查看更多饰品槽
- 支持对应饰品渲染开关

### 4. 卡槽箱

当终端存在对应工具箱/卡槽箱能力时显示。

用途：

- 主要用于 AE/相关模组升级卡的集中放置
- 名称已从早期“工具箱”调整为“卡槽箱”，避免与工具包混淆

### 5. 工具包

始终显示。

功能定位：

- 存放各种工具类物品，而不是升级卡
- 默认支持不可堆叠工具
- 支持独立滚动面板
- 支持快捷键直接打开
- 若把 AE 网络工具放入工具包，WCWT 已额外处理其访问逻辑

前 11 个专用槽位主要面向：

- 剑
- 镐
- 斧
- 锹
- 锄
- 扳手
- AE 石英切割刀
- AE 网络工具
- AE 内存卡
- Mekanism 配置卡
- Mekanism 配置器

服务端可配置：

- 工具包槽位数量
- 当前范围：`11 ~ 640`

### 6. 谐振过载编码器

始终显示，但其完整功能依赖对应模组。

#### 过载样板编辑区

主要整合：

- `AE2 Lightning Tech`

功能：

- 选中样板缓存区里的样板后，查看输入/输出条目
- 切换匹配模式
- 将样板转换为过载样板

#### 谐振样板转换区

主要整合：

- `AE2 Crystal Science`

功能：

- 批量扫描样板缓存区中的处理样板
- 转换为谐振样板
- 转换结果放入谐振样板缓存区
- 谐振缓存区满后停止继续转换

## 快捷键

所有扩展 UI 都提供独立快捷键绑定，默认可在按键设置中修改：

- 打开高级编码
- 打开装饰盔甲
- 打开饰品栏
- 打开卡槽箱
- 打开工具包
- 打开谐振过载编码器

特殊说明：

- 工具包快捷键支持在**未打开终端界面时**直接唤起
- 其它扩展 UI 快捷键通常在终端界面已打开时生效

## 模组联动说明

### AE2 Wireless Terminal Library

提供或复用的能力包括：

- 无线终端基础行为
- 无线终端设置
- 磁卡菜单
- 无线访问点联动

### AdvancedAE

主要用于：

- 高级样板方向编辑
- 便携元件工作台相关体验

### ExtendedAE / ExtendedAE Plus

主要用于：

- 样板供应器管理
- 样板上传
- 定位供应器
- 打开供应器目标 UI
- 搜索映射逻辑

### NeoECOAEExtension

如果网络中存在 ECO 合成子系统，WCWT 会优先尝试把部分样板上传到 ECO。

### EMI / JEI

WCWT 对两者都做了编码区填充兼容，支持将支持的配方材料拉入编码区。
