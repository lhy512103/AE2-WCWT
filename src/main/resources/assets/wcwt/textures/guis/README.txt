纹理文件说明
=================

此目录应该包含以下纹理文件：

1. wcwt_states.png (256x256)
   - 包含所有按钮和UI状态的纹理
   - 合成锁定按钮：(2,0,12,15), (18,0,12,15), (34,0,12,15)
   - 扩展UI按钮底图：(0,128,28,27)
   - 扩展UI按钮：(128,128,20,17), (160,128,20,17)
   - 样板倍增按钮：(0,160,34,30), (64,160,34,30)
   - 显示开关图标：(192,0,5,5), (208,0,5,5)

2. wcwt_advanced_coding.png (256x256)
   - 高级编码面板背景 (134x186)
   
3. wcwt_cosmetic_armor.png (256x256)
   - 装饰装甲面板背景 (80x38)
   
4. wcwt_curios.png (256x256)
   - Curios面板背景 (134x186)

注意：这些纹理文件需要从对应的附属模组中提取或重新绘制。
工具箱面板使用AE2的纹理：ae2:textures/guis/extra_panels.png
