# ME Wireless Comprehensive Work Terminal

The ME Wireless Comprehensive Work Terminal (WCWT) combines ME storage access, manual workstations, pattern encoding, pattern-provider management, equipment storage, and several AE2 addon integrations in one portable screen.

<ItemImage id="wcwt:wireless_comprehensive_work_terminal" scale="3"></ItemImage>

## Before You Start

1. Charge the terminal and link it to a security terminal and ME network like any other AE2 wireless terminal.
2. Bind **Open ME Comprehensive Work Terminal** in Controls. WCWT has its own key and does not reuse another wireless terminal key.
3. Once inside the terminal, use the title-bar help button to return to this guide.
4. Extension features require their matching upgrade card in the terminal extension slots. Installing an addon without installing its card does not open the panel.

## Feature Requirements at a Glance

- Basic storage, manual crafting, pattern encoding, and pattern management: AE2 only.
- Advanced-pattern input sides: **Advanced AE**.
- Bulk-cell compression cutoff: **MEGA Cells**, plus a compatible cell, compression card, and compression chain.
- Overload patterns: **AE2 Lightning Tech**.
- Resonating patterns: **AE2 Crystal Science**.
- Accessory slots: **Curios API**; cosmetic armor slots require **Cosmetic Armor Reworked**.
- Recipe transfer and alternative ingredients: **JEI or EMI**; conflicting recipes can integrate with **Polymorph**.
- Special materials such as Mekanism chemicals and lightning also require the source mod and compatible AE2 storage support.
