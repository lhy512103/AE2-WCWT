# 过载与谐振样板

谐振过载编码卡会根据已安装模组显示不同区域。安装任意一个支持模组即可显示卡片；只显示当前实际可用的转换功能。

## 过载样板编辑器

**需要：AE2 Lightning Tech（模组 ID `ae2lt`）。**

1. 在主界面缓存区放入可解码的处理样板。
2. 打开谐振过载编码面板，并在缓存区左键选中该样板。
3. 上方列表显示所有非空输入和输出。
4. 点击每一行的匹配开关，在严格匹配与仅 ID 匹配之间切换。
5. 每次切换会立即提交转换：普通 AE2 处理样板会在原缓存槽中变为过载样板，已有过载样板则原位更新。

严格匹配会比较完整数据组件；仅 ID 匹配会忽略组件差异。对于带蜂种、药水、附魔、耐久或其它 NBT/组件语义的材料，建议保持严格匹配，避免机器接受错误变体。

转换不需要空白样板，也不会把结果放入其它槽。未选中样板、样板不可解码或不是支持的处理格式时不会执行。

## 谐振样板转换器

**需要：AE2 Crystal Science（模组 ID `ae2cs`）。**

1. 把需要转换的处理样板放入样板缓存区。
2. 确认面板中的 21 格谐振样板存储区有空位。
3. 点击 **转换为谐振样板**。
4. 转换器按缓存槽顺序处理所有支持的样板，而不是只处理当前选中的一个。
5. 每个成功结果进入谐振存储区的第一个空槽，同时从来源缓存槽消耗一个样板。

不支持的样板会跳过。谐振存储区满时转换停止并给出提示，尚未成功转换的来源样板会保留。操作前请确认空位，避免误以为按钮只转换当前选择。

## 两种转换的区别

- 过载编辑：只处理当前选中样板，结果原位覆盖，不消耗空白样板。
- 谐振转换：批量处理缓存区，结果进入独立 21 格存储区，每个成功结果消耗一个来源样板。
- 两者都依赖附属模组自己的样板格式和编码能力；WCWT 只提供集中界面和安全的数据传递。
