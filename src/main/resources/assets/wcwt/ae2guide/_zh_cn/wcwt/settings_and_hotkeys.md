# 设置、快捷键与排错

## 客户端设置

客户端选项控制界面显示和只影响本地玩家的行为，例如：中键选取、缺少现货时发起合成、自动补货、磁力拾取、拾取到 ME 或玩家、候选材料收藏优先、手动替换模式和供应器搜索框自动填充。

关闭“编码样板填充供应器搜索”后，普通编码不会改写搜索框；若样板上传功能本身处于开启状态，上传流程仍可填充搜索内容以完成目标匹配。

## 快捷键

在 Minecraft 按键设置中搜索 WCWT，可绑定打开主终端和各扩展面板的快捷键。铁砧改名框、搜索框和映射编辑框获得焦点时，文字输入优先，不会触发终端功能快捷键。

若快捷键冲突，请在控制设置中移除重复绑定；不要依赖其它 AE2 无线终端的打开键代替 WCWT 独立按键。

## 服务端 OP 指令

只有 OP 可读取或修改以下 WCWT 服务端选项：

- `/wcwt config toolkitSlotCount <数值>`：设置工具包通用/记忆槽数量限制。
- `/wcwt config patternProviderActiveRefresh <true|false>`：控制样板供应器主动刷新行为。

不带合法值或权限不足时不会修改服务器配置。专用服务器配置与客户端供应器映射文件相互独立。

## 常见排错顺序

1. **按钮不显示**：确认对应附属模组已加载，并把所需扩展卡放入终端。
2. **样板不能转换**：确认它是支持的已编码处理样板，而不是空白、合成、锻造或切石样板。
3. **特殊材料拖不进去**：确认 JEI/EMI、材料来源模组和 AE2 对应存储类型均已加载。
4. **供应器打不开**：检查区块、维度、方块朝向、目标容器和玩家权限。
5. **流体替换不灌装**：检查网络流体数量、空容器是否返还到原格，以及容器是否支持标准流体处理。
6. **列表图标短暂回滚**：服务器拒绝了槽位操作；检查权限、距离、网络重构或目标槽内容。
7. **整合包行为与测试档不同**：先查看 `latest.log` 中缺少的可选模组、版本不一致、资源包覆盖或兼容模组拦截信息。

对专用服务器问题，客户端界面显示正常并不代表服务器接受了操作。应同时检查客户端和服务端日志，并以服务器槽位状态为最终结果。
