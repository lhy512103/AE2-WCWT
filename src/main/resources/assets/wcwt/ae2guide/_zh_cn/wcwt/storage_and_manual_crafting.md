# 存储与手动工作区

## ME 存储区

物品、流体和其它 AEKey 类型使用同一套 AE2 网络访问。顶部类型按钮用于限制当前显示内容，排序、显示方式和每行数量沿用 AE2 终端规则。

- 中键选取方块会从 ME 网络请求准星指向的方块，并自动选中实际收到物品的快捷栏格。
- 收藏条目便于查找；启用“优先收藏候选”后，JEI/EMI 多候选配方会优先选用收藏材料。
- 带 NBT 或数据组件的同 ID 物品会精确区分，例如不同 Productive Bees 蜜蜂刷怪蛋。

## 工作台模式

1. 切换到 3×3 工作台。
2. 手动放入材料，或在 JEI/EMI 配方页点击传输按钮。
3. 配方预览锁定时，终端会检查网络和玩家物品栏中的可用数量。
4. 从输出槽取走结果。玩家物品栏中被计入的材料会先实际移动到合成格，不会只参与数量计算。

按住 Shift 拉取一组配方用量仅对非合成配方物品生效；可合成物品仍按 AE2 的合成流程处理。

## 物品替换

物品替换只属于手动工作区，与样板编码区的同名按钮互相独立。

1. 在工作台模式开启 **物品替换**。
2. 通过 JEI/EMI 传入含标签或多候选 Ingredient 的配方。
3. 正常连续合成。当当前候选耗尽时，终端只在替换模式开启期间查询该配方的其它有效候选。
4. 若网络或玩家物品栏存在另一种符合配方的材料，会继续补入合成格。

耐久物品和带数据组件物品仍会执行配方合法性校验；不会用“同 ID”粗略替换。关闭替换后不会进行额外候选查找。

## 流体替换

流体替换用于“容器参与配方，合成后返还空容器”的连续合成，例如水桶参与配方后返还空桶。

1. 在工作台模式开启 **流体替换**。
2. 保证 ME 网络中有对应流体，并有 AE2 可访问的流体存储。
3. 使用装有该流体的容器开始合成。
4. 合成返还空容器后，终端尝试从网络提取所需流体并重新灌装，再放回对应合成格。

该功能不会凭空创建容器或流体。容器必须支持标准流体处理，网络也必须有足量流体；模组自定义且不暴露标准灌装行为的容器可能不受支持。

## 锻造台、铁砧与切石机

切换模式后，滚轮传输和快捷移动会以当前可见工作区为目标，不会再写入隐藏的工作台格。

- 锻造台：按模板、基础物品和附加材料槽工作。
- 铁砧：放入输入物品和材料，在改名框中输入新名称后取出结果。编辑名称时，终端快捷键不会抢占键盘输入。
- 切石机：放入输入并选择输出配方。

这些模式不使用工作台的物品替换和流体替换逻辑。
