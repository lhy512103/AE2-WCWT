# 样板编码

## 编码合成样板

1. 打开样板编码区并选择合成模式。
2. 放入空白样板；编辑已有样板时可直接放入已编码样板。
3. 手动摆放配方，或从 JEI/EMI 配方页传输。
4. 冲突配方安装 Polymorph 后可在配方选择控件中指定实际结果。
5. 检查输出预览，点击编码。

结果会优先写入编码槽；根据当前界面操作，也可进入样板缓存区或启用的供应器上传流程。配方传输会保留 NBT 和数据组件，不会把同 ID 的特殊变体随意替换。

## 编码处理样板

1. 切换到处理模式。
2. 在输入侧放入设备要消耗的材料，在输出侧放入预期产物。
3. 使用数量编辑和倍增功能设置每次处理的数量。
4. 需要时开启合并材料、物品替换或流体替换标记。合并材料状态会保存在当前终端物品上，关闭并重新打开终端后仍保持。
5. 点击编码并将样板放入对应机器的样板供应器。

处理样板允许物品、流体以及 AE2 已注册的其它 AEKey 类型。JEI/EMI 可拖入的特殊材料会先通过 WCWT 的 AE2 通用类型转换，因此 Mekanism 化学品和 Lightning Tech 闪电等类型可进入高级编辑区；前提是对应模组与存储类型支持已加载。

## 锻造与切石样板

选择锻造台或切石模式，传入配方并编码即可。它们仍是各自的配方样板，不支持 Advanced AE 的处理样板输入方向转换。

## 编码区替换开关

样板编码区的 **物品替换** 和 **流体替换** 是写入样板或编码流程的选项，不与手动合成区同步。两组按钮会分别保存各自状态。

## 缓存区与 SCALE

样板缓存区是批量上传、高级编码、复制替换和特殊样板转换的共享工作区。

- 先把已编码样板放入缓存区，再在相应面板中左键选中。
- `SCALE` 可对缓存样板数量执行乘倍或除倍；结果仍写回缓存中的原样板位置。
- 倍增不会使非法数量变为合法配方；超出样板或 AEKey 数量限制时会拒绝或截断。
- 缓存区不是供应器。完成编辑后仍需手动移动或使用样板管理上传。
