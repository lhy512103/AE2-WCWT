# 高级编码与元件编辑

高级编码面板包含四组互相独立的工具：Advanced AE 输入方向、样板复制、批量材料替换和存储元件工作台。

## 选择待编辑样板

1. 把已编码样板放入主界面的样板缓存区。
2. 打开高级编码面板。
3. 左键点击缓存槽中的样板。这里是“选择”，不会把样板拿到鼠标上。
4. 面板读取所选样板；换选其它缓存槽后，方向和复制来源也随之切换。

无法解码、空白或当前工具不支持的样板不会显示可执行内容。

## 输入输出方向编辑

**需要：Advanced AE。** 此功能接入 Advanced AE 的高级处理样板格式。

1. 选择一个普通 AE2 处理样板，或已有的 Advanced AE 高级处理样板。
2. 面板按输入 AEKey 列出材料行。每行右侧显示当前允许进入的方向。
3. 点击方向控件，在任意、北、东、南、西、上、下之间切换。
4. 第一次成功修改普通处理样板时，WCWT 会调用 Advanced AE 编码器，把它转换为高级处理样板。
5. 转换后的样板直接写回原来的缓存槽，不消耗空白样板，也不会生成第二份。

“任意”表示不限制该输入的供应面。方向用于支持该格式的机器或供应器解释输入路线，并不是让 WCWT 在世界中自动旋转方块。

合成、锻造和切石样板不属于处理样板，不能用此功能转换。若 Advanced AE 未加载，方向编辑区不会提供转换操作。

## 复制按钮

1. 在缓存区选中要复制的已编码样板。
2. 在复制输入槽准备一个空白样板；若允许网络补充且网络有空白样板，面板会尝试自动填入。
3. 点击复制。
4. 新样板进入真实复制结果槽；若槽内已有完全相同的样板，可继续堆叠。

复制只读取当前选中的一个样板，不会修改来源，也不会复制整个缓存区。结果槽满、缺少空白样板或来源不可解码时不会执行。

## 批量替换按钮

替换区的两个槽是幽灵条件，不会吞掉鼠标上的真实材料。

1. 把“要查找的材料”拖入左侧幽灵槽。
2. 把“替换成的材料”拖入右侧幽灵槽；右侧留空表示从样板中删除该材料。
3. 点击替换。
4. WCWT 扫描样板缓存区中的所有可解码样板，并把成功重编码的结果写回各自原槽。

处理样板可替换物品、流体和其它通用 AEKey。合成样板只能使用物品候选，而且替换后必须仍能通过原配方和输出校验。无法重编码或会改变为无效配方的样板会跳过，不会强行破坏。

JEI/EMI 中的 Mekanism 化学品、Lightning Tech 闪电等特殊类型可拖入幽灵槽；这依赖对应模组注册的 AEKey/GenericStack 转换支持。

## 存储元件工作台

1. 把真实存储元件放入元件槽。
2. 右侧升级槽会按该元件实际支持数量显示，放入压缩卡等升级。
3. 在 3×6 可滚动配置网格中点击或从 JEI/EMI 拖入过滤内容。这里是幽灵分区配置，不消耗物品。
4. **按存储内容分区** 会读取元件当前已有类型并填入配置；**清除** 会清空全部配置。
5. 复制模式按钮控制取下元件时是否保留当前编辑配置，行为与 AE2 元件工作台的复制模式一致。

## MEGA Cells 大宗压缩截断

**需要：MEGA Cells。** 按钮仅在以下条件同时满足时出现：放入兼容大宗元件、元件启用压缩能力、安装所需压缩卡，并且当前标记物品存在可识别压缩链。

按钮位于复制模式按钮右侧。左键和右键以相反方向循环可用截断级别，例如在普通圆石和 AllTheCompressed 提供的多重压缩圆石之间选择。设置只作用于当前元件及其集成配置，不会扫描或改写整个 ME 网络。

按钮不出现时，请检查元件类型、升级卡、被标记物品是否确有压缩链，以及 MEGA Cells 是否加载；只安装 AllTheCompressed 并不足以启用该功能。
