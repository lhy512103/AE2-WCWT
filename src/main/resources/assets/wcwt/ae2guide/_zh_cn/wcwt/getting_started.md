# 入门与界面

## 主界面分区

- 左侧为 ME 存储区：搜索、排序、存取、收藏和发起合成。
- 中下部为玩家物品栏与快捷栏。
- 右侧主工作区可切换手动合成、样板编码、样板缓存和样板管理。
- 界面边缘的工具栏用于设置、显示元件、扩展面板和其它快捷功能。

按钮是否出现由当前升级卡、已安装模组和当前工作模式共同决定。看不到某个面板时，先检查附属模组是否已加载，再检查终端扩展升级槽中是否安装了对应卡片。

## 第一次使用

1. 打开终端，确认左侧能显示当前 ME 网络内容。
2. 在搜索框中输入物品名、模组名或 AE2 支持的搜索表达式。
3. 左键或右键网络条目进行常规存取；按住 Shift 可在玩家物品栏与当前区域间快速移动。
4. 网络没有现货但已有可用合成 CPU 和样板时，点击可合成条目进入 AE2 合成数量界面。
5. 点击右侧模式按钮进入需要的工作区；返回按钮会回到主终端而不会关闭整个界面。

## 无线连接异常

若存储区为空或提示无法连接，请依次检查：终端电量、无线接入点覆盖、量子桥两端网络状态、安全终端权限、维度限制和 AE2 无线终端配置。WCWT 支持仅通过量子桥连接的网络，但量子桥本身必须在线。

## 界面状态保存

客户端会保存多数显示偏好、面板开关和手动合成替换模式。服务端选项由服务器管理；在专用服务器中，只有 OP 可通过 `/wcwt config` 修改指定服务端选项。
