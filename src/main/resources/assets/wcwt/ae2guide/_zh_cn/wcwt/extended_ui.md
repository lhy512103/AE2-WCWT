# 扩展面板总览

扩展面板通过终端扩展升级卡启用。安装对应模组后，支持该模组的升级卡才会出现在物品列表和配方查看器中；把卡放入终端后，相应按钮才会显示。

## 面板与升级卡

- **高级编码卡**：高级样板方向编辑、样板复制、批量替换和存储元件工作台。
- **谐振过载编码卡**：安装 AE2 Lightning Tech 或 AE2 Crystal Science 后显示对应转换区。
- **饰品栏卡**：需要 Curios API。
- **装饰盔甲卡**：需要 Cosmetic Armor Reworked。
- **卡槽箱卡**：集中保存受支持的升级卡。
- **工具包卡**：保存工具并提供记忆槽。

扩展 UI 可以通过右侧按钮或已绑定快捷键打开。离开面板不会丢弃真实槽位中的物品；幽灵配置槽只保存过滤条件，不代表其中存有真实物品。

## 延迟与刷新

WCWT 仅在面板打开、按钮启用或玩家执行操作时请求相应数据。高级替换候选、元件配置和供应器列表不会在无关界面持续全量扫描。专用服务器网络较慢时，真实槽位仍以服务器确认为准。

后续章节分别说明高级编码、特殊样板、装备和工具包的完整操作。
